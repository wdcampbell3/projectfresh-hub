## ï¿½ Tips for Talking to Your AI Assistant

### âœ… DO: Be Specific

**Good:** "In the Pong game, make the ball speed increase by 10% each time it hits a paddle"

**Not as good:** "Make Pong harder"

### âœ… DO: Start Small

Instead of asking for a complete new game all at once, build it piece by piece:

1. "Create a basic canvas with a bouncing ball"
2. "Add keyboard controls to move a paddle"
3. "Add collision detection between the ball and paddle"
4. "Add a score counter"

### âœ… DO: Test Often

After each change, refresh your browser to see the result. If something breaks, tell your AI:

> "That change broke the game - I see this error: [paste the error]. Can you fix it?"

### âŒ DON'T: Ask for Too Much at Once

The AI works best with focused requests. Instead of "build me a complete RPG with inventory, combat, and quests," start with one feature at a time.

### âŒ DON'T: Worry About Mistakes

If something goes wrong, you can always ask:

> "Undo the last change you made"

or

> "Reset the file back to how it was before we started"

The AI will walk you through:

2. Connecting your local code to GitHub
3. Uploading (pushing) your code

### Step 3: Connect to Netlify

1. Log into [netlify.com](https://netlify.com)
2. Click **"Add new site"** â†’ **"Import an existing project"**
3. Choose **GitHub** and select your repository
4. Use these settings:
   - **Build command:** `npm run build`
   - **Publish directory:** `build`
5. Click **Deploy!**

Your games will be live at `https://something-random.netlify.app` in about 2 minutes!
